Mars Web Scraping

|

Skills:

  • Python / Flask / Pandas
  • Requests
  • HTML / CSS / Bootstrap
  • Data Parsing (local JSON)
  • Javascript
  • Web Scraping using Splinter, BeautifulSoup

This Python-Flask app was created to demonstrate web-scraping ability by creating a web-scraping Python script to scrape information from mutliple Mars-related websites. The python script employs BeautifulSoup and Splinter to visit the websites and automatically collect and display relevant information, as well as collecting images to display within the app. The information is then placed into a MongoDB database which is accessed in order to display the results. The app can be modified to run daily, checking for latest news and placing the results into a database to track the results over time. The scraping functionality has been removed from the app due to lack of resources necessary for full functionality.

The follwing websites are used in the collecting of this information:

  • redplanetscience.com
  • spaceimages-mars.com
  • galaxyfacts-mars.com
  • marshemispheres.com
